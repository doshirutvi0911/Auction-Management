# 🏷️ Auction Management System (Java)

## 📌 Overview
The **Auction Management System** is a console-based application developed using **Java SE**. It simulates a real-world online auction platform where multiple users interact, including **Owner, Seller, and Buyers**.

This project demonstrates core **Object-Oriented Programming (OOP)** principles and provides a complete auction workflow from item listing to bidding and payment.

---

## 🚀 Features

- 👤 **Multi-User Roles**
  - Owner: Manages sellers, items, and earnings
  - Seller: Handles auction items and views profits
  - Buyer: Registers, bids, and completes payments

- 🔐 **Secure Login System**
  - Strong password validation
  - Role-based authentication

- 💰 **Auction Process**
  - Add items → Place bids → Select winner → Payment

- 📊 **Revenue Distribution**
  - Seller: 5%
  - Owner: 95%

- ✅ **Bid Validation**
  - Prevents invalid or low bids
